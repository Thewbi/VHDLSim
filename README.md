# VHDLSim
VHDL Simulator

Following versions of VHDL exist:

VHDL-87 (1987, Publication of IEEE Standard)
VHDL-93 (Revised standard (named VHDL 1076-1993))
VHDL-2002 (Revised standard (named VHDL 1076-2002))
VHDL-2008
VHDL-2019

This simulator supports VHDL-2008.
